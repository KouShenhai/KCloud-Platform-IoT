= 边缘网关

== 下载依赖

[source,shell]
----
# node版本必须大于或等于18.x
npm install -g pnpm
pnpm install
----

== 运行项目

[source,shell]
----
pnpm dev
----

== 部署项目

[source,shell]
----
pnpm build
----

== User Management

The `/users` page is available only to users whose JWT payload contains
`role=admin`. Operator users can continue to use authenticated gateway pages but
will not pass the `canManageUsers` access check for user administration.

== Menu Management

The `/menus` page is available only to users whose JWT payload contains
`role=admin`. It manages the edge gateway menu tree with name, path, icon,
parent menu, sort order, and enabled or disabled status.
